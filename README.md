# CMMC Test Simulator

A simple web app for CMMC exam preparation.

![CMMC Test Simulator](screenshots/main_screen.png)

## Features

- **Practice Mode**: Study with immediate feedback
- **Simulated Test**: Full exam experience  
- **Rapid Memory**: Flashcard review
- **Dark Mode**: Easy on the eyes
- **Progress Tracking**: Visual progress indicators

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

## Questions

206 practice questions covering all CMMC domains.

## Tech Stack

- React + Vite
- Tailwind CSS
- No external APIs
